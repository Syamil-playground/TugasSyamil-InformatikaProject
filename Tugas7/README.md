# Praktikum 7 — PHP Dasar

## Struktur Folder
```
lab7_php_dasar/
├── php_dasar.php
├── latihan_get.php
├── latihan_post.php
├── tugas.php
└── README.md
```

## Langkah Pengerjaan
1. Jalankan Apache di XAMPP.
2. Simpan folder ini ke dalam `htdocs`.
3. Akses di browser:
   - http://localhost/lab7_php_dasar/php_dasar.php
   - http://localhost/lab7_php_dasar/latihan_get.php?nama=Agung
   - http://localhost/lab7_php_dasar/latihan_post.php
   - http://localhost/lab7_php_dasar/tugas.php
4. Isi form dan lihat hasilnya di browser.

## Deskripsi Singkat
Latihan ini memperkenalkan dasar pemrograman PHP:
- Variabel dan echo
- Form `GET` dan `POST`
- Struktur kontrol (if, switch, loop)
- Perhitungan umur & gaji berdasarkan pekerjaan
